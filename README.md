# RequestSCP
Switch the player's SCP Role by yourself.

# English

A plugin based on Exiled for SCP:SL. Switch the player's SCP Role by yourself.

Usage:

Step 1. Press ~
Step 2. input command .request scp
Step 3. Press Enter

Like:

.request 106

And your role will be 106.

# Chinese

本插件基于 Exiled. 本插件允许你自己切换角色。

用法:

1. 按 ~
2. 输入 .request scp
3. 按下回车键

如：

.request 106

然后你的角色就是106了。
